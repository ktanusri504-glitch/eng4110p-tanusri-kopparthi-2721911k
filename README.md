# ENG4110P Wind Turbine Tower Project

This repository contains the Python code and processed Excel data used in support of the final year project report on wind turbine tower response under uncertain wind and gust loading.

Contents:
- Abaqus_Python.py: Python script used for the automated Abaqus simulation workflow
- tapered_no_gust_100.xlsx: processed simulation outputs for the no-gust Monte Carlo cases
- tapered_all_gust_100.xlsx: processed simulation outputs for the gust Monte Carlo cases

These materials are provided for academic reference in support of the results presented in the report.
